Documentation

Installation

In order to install and use this tool, simply upload it on your server, where run the joomla db where you want insert the new users from CSV file.

Tool Usage

Using this tool is very simple! Just follow this steps:on.

  • Once the tool is uploaded on server, upload your CSV file inside the dir joomla_users_importer/csv (where you find an example CSV file named userlist.csv)
  • Launch tool from the browser http://yourserver.xx/path/to/joomla_users_importer
  • Just fill all form fields then press the "Import Users" button.
    See below how to fill fields correctly

Remember that the "logs" directory must be writable by the webserver!

Form Fields

The tool form is divided in 3 different rows.
The first row is dedicated to MySQL database connection, the second row is for the users settings and the last row is for the CSV filename.

DB Fields

  • DB Host: your mysql server host (probably "localhost")
  • DB Name: your joomla database name
  • DB Prefix: the prefix of your joomla database tables (without _)
  • DB UserName: your joomla database username
  • DB Password: your joomla database password

Users Fields

  • DB Users Group: choose a group for all users
    Remember that if you choose "none" option, users will be not added to any group.
  • DB Users Blocked: choosing "Yes" all users are inserted as blocked, so you need to unblock them one by one from the joomla admin panel
  • Users Activation: choosing "No" all your new users will be inserted as already active users. Choosing "Yes" all users must be activated one by one from the joomla admin panel
  • Users Send Email: coosing "Yes" all users will recive system emails.
  • Users Require Pswd Reset: coosing "Yes" all users will need a password rest at first login.

CSV Field

  • CSV File: place here your csv file name. Remember that CSV file must be placed inside "csv" directory

CSV File

Your user CSV file must be formatted as shown below:


Name,Surname,email@emaildomain.xx
Name1,Surname1,email1@emaildomain.xx
Name2,Surname2,email2@emaildomain.xx
Name3,Surname3,email3@emaildomain.xx
